Welcome to librarymanagementsystem-lending Service

Handles book lending lifecycle: checkouts, returns, renewals, member and librarian transactions, reservations (holds), overdue detection/events; tracks per-user and per-branch activity; provides real-time lending state for library network.

Available Routes

  • getLoan => GET /loans/:loanId
  • createLoan => POST /loans
  • updateLoan => PATCH /loans/:loanId
  • deleteLoan => DELETE /loans/:loanId
  • listLoans => GET /loans
  • getReservation => GET /reservations/:reservationId
  • createReservation => POST /reservations
  • updateReservation => PATCH /reservations/:reservationId
  • deleteReservation => DELETE /reservations/:reservationId
  • listReservations => GET /reservations
  • getLoanEvent => GET /loanevents/:loanEventId
  • createLoanEvent => POST /loanevents
  • updateLoanEvent => PATCH /loanevents/:loanEventId
  • deleteLoanEvent => DELETE /loanevents/:loanEventId
  • listLoanEvents => GET /loanevents
